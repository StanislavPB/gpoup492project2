Метод createProject

Метод createProject предназначен для создания нового проекта и добавления его в репозиторий (хранилище данных).
Он выполняет проверку на наличие дубликатов по идентификатору проекта и добавляет проект только в случае,
если проект с таким идентификатором еще не существует.

Параметр метода:
- Метод принимает объект класса Project в качестве параметра, который должен быть добавлен в хранилище (репозиторий).

Проверка на дубликат:
- Сначала метод проверяет, существует ли уже проект с таким же идентификатором (ID).
- Метод isPresent() используется для проверки, существует ли проект с таким идентификатором в репозитории.

Добавление проекта в репозиторий:
- Если проект с таким ID не найден (т.е. getProjectById(project.getId()) возвращает пустое значение),
  метод вызывает addProject у репозитория, чтобы добавить новый проект в хранилище.

Возвращаемое значение:
= Если проект успешно добавлен, метод возвращает true, указывая на успешное создание проекта.
